## 🛠️ The 3-Step Audit Process

### 1. Diagnostic Phase
Use tools to crawl the site and identify 404s, broken links, and redirect loops.
> *Action:* "Analyze [URL] for broken links and canonical tag issues."

### 2. Gap Analysis
Compare the site with the top 3 competitors in the SERP.
> *Action:* "Identify top 5 keywords [Competitor URL] is ranking for that [Your URL] is missing."

### 3. Authority Building
Focus on high-DA outreach rather than mass directory listing.
> *Action:* "Generate a 12-week outreach roadmap for [Niche] based on competitor referring domains."
